Natural Numbers

Informal description

A natural number is the size of a group of objects. For example, a group with a single object has size "one", and a group with two objects has size "two". Given a natural number, the next natural number is then the immediate bigger size. For example, the next natural number to one is two, and the next of which is three. Some authors include the number "zero", the size of a group with nothing in it, as a natural number. This website uses the definition that doesn't include zero.
