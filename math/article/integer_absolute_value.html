Informal description

The absolute value of an integer is a nonnegative integer. The absolute value of a nonnegative integer is itself, and the absolute value of the negative integer \(-a\) is \(a\).

Motivation

Negative integers have positive "sizes", for example \(3-5\) has the size of \(5-3=2\). The "size" of a nonnegative integer is itself, and the "size" of the negative integer \(-a\) is \(a\).
