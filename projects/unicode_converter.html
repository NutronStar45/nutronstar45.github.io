Instructions
Clicking the "Convert" button next to a representation converts from that representation.
Code points and decimal inputs must be separated by whitespaces.
Whitespaces are not needed for encoding forms in hexadecimal and binary.
Convert on enter: pressing enter in a representation input box converts from that representation.
Exclude text: if "Convert on enter" is enabled, pressing enter in the "Text" input box doesn't trigger conversion.
