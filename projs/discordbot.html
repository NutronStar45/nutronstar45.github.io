About Discord

Discord is a VoIP, instant messaging and digital distribution platform designed for creating communities.
Users communicate with voice calls, video calls, text messaging, media and files in private chats or as part of communities called "servers".
Servers are a collection of persistent chat rooms and voice chat channels.
Discord runs on Windows, macOS, Android, iOS, iPadOS, Linux, and in web browsers.
Discord on Wikipedia

About Discord bot

Discord bot is a Discord API use to automate processes using bots.
Commonly used on

  • Reaction rolesAutomatically gives or takes a role when user reacted a message (emoji)
    Example: Reaction Roles
  • MusicSearch and play music directly from the Internet, may offers downloadable files
    Example: Groovy
  • Server greetingSend a message to the server and/or DM the user when the user joined the server
    Example: MEE6 Server Greeting mod
  • Auto moderationBlacklist mentions, links or certain phrases like swear words from sending
    Example: MEE6 Moderation mod

How to get it working

In this section, we will go through the steps to make the bot working.

How to get Discord

These steps are all tested in Windows except the MacOS installation. Not sure if different platforms use different method.
Check the article on For additional information for methods on different platforms.

You can use Discord directly in your browser or install the app into your device.

Use directly in browser

Follow these steps if you haven't registered on Discord:

  1. Go to the registry page
  2. Fill out the form and click Continue
  3. Check if there's an email sent from Discord. If not, click Resend on the top of the registry page
  4. Click Verify Email on the mail you received
    5. For additional information, see this article

If you have already registered, follow these steps to login:

  1. Go to the login page
  2. Fill out the form and click Login
    3. For additional information, see this article

Install the app

Follow these steps to install Discord on MacOS:

  1. Go to the download page
  2. Open the downloaded file Discord.dmg
  3. Drag the Discord logo and drop it over to Applications folder

Follow these steps to install Discord on Windows:

  1. Go to the download page
  2. Click the Download Now button
  3. Open the downloaded file DiscordSetup.exe

For additional information, see this article
