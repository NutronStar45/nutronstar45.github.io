Nim

About

Nim is a game in which two players take turns removing objects from distinct heaps or piles. [wiki]

Gameplay

The game starts with two or more heaps, and every heap contains at least one object.
Player 1 takes one or more objects from one heap, Player 2 takes one or more objects from one heap, and so on.
In a normal play, the player who picks up the last object wins; in a misère play, the player who picks up the last object loses.

How to Play

The available game options are: Type, Mode, Heap Sizes, Difficulty (PvC only), and First (PvC only).
Type: PvP (Player vs Player) or PvC (Player vs Computer).
Mode: in a normal play, the player who picks up the last object wins; in a misère play, the player who picks up the last object loses.
Heap Sizes: Determines the sizes of the heaps, in the format of 3,4,5.
Difficulty (PvC only): determines how difficult you want the game to be, options are: Easy, Medium, Hard, and Extreme.
First (PvC only): determines the first player.
